# B2B Admin Dashboard

A modern admin dashboard for managing B2B business operations.

**Status:** Work in Progress

## About the Project
This project is currently under active development. The goal is to build a clean, fast, and scalable admin panel for B2B business management.

## Planned Features
- Dashboard analytics
- Order management
- Product management
- Sales tracking
- Customer Management
